Silent Distress Detector (SSD)

Detecting Unnoticed Personal Emergencies Using Sensor Intelligence

View Demo

Problem Statement

Many individuals such as elderly people, hostel students, and night-shift workers face emergencies where they are unable to call for help. Existing systems rely on manual SOS or voice input, which fails in silent distress situations.

Proposed Solution

Silent Distress Detector (SSD) is a software-based system that passively monitors abnormal inactivity using smartphone sensors and alerts trusted contacts if the user does not respond.
