Silent Distress Detector (SSD)

Detecting Unnoticed Personal Emergencies Using Sensor Intelligence

View Demo

Problem Statement

Many individuals face personal emergencies where they are unable to call for help. Existing systems rely on manual input, which fails in silent distress situations.

Proposed Solution

SSD passively monitors abnormal inactivity using existing smartphone sensors and automatically alerts trusted contacts if the user fails to respond.
